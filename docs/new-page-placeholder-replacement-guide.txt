~ For best SEO and social sharing, you should use a high-quality, relevant image for {{OG_IMAGE}}, {{TWITTER_IMAGE}}, and {{SCHEMA_IMAGE}}. Here are the best practices:
  @ The same image can be used for all three placeholders if it fits the requirements.
  @ Recommended size: 1200x630 pixels (landscape, 1.91:1 aspect ratio).
  @ Format: JPG or PNG.
  @ The image should visually represent the page or your brand (e.g., a project screenshot, your logo, or a featured graphic).
  @ The image URL should be absolute (e.g., https://iknittheweb.com/src/img/social/your-image.jpg).
  @ Example:
  %  {{OG_IMAGE}}: https://iknittheweb.com/src/img/social/portfolio-preview.jpg
  %  {{TWITTER_IMAGE}}: https://iknittheweb.com/src/img/social/portfolio-preview.jpg
  %  {{SCHEMA_IMAGE}}: https://iknittheweb.com/src/img/social/portfolio-preview.jpg
  @ If you want to use different images for each, you can, but for most pages, using the same well-chosen image is perfectly fine. Just make sure the image is accessible and optimized for web.
